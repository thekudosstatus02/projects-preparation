What is Kernal? (V. Imp)
ans : Kernel is central component of an operating system that manages operations of computer and hardware. 
It basically manages operations of memory and CPU time. It is core component of an operating system
The major aim of kernel is to manage communication between software i.e. user-level applications and hardware i.e., CPU and disk memory.
